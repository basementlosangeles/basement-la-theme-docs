# Editing "You May Also Like"

<span class="bla-badge">Product</span>

**What it changes:** The "You may also like" carousel on product pages — its heading,
the "no products" fallback text, and how far the arrows scroll.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template to a **Product** page → click the **Related Products** section.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open a **Product** template and select the **Related Products** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Edit the **Title Text** (default "You may also like").</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Adjust **Arrow Nav Button Scroll Amount** if the carousel scrolls too far or too little.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "Which products show is set per product"
    The actual recommended products come from each product's *Related products*
    metafield in **Shopify Admin → Products**, not from this section. Here you only
    edit the wording and behavior.

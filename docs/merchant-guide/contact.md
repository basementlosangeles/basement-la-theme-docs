# Editing the Contact Page

<span class="bla-badge">Pages</span>

**What it changes:** The Contact page heading, intro text, the form field placeholders,
and the submit button wording.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template to the **Contact** page → click the **Contact** section.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open the **Contact** page in the template dropdown and select the **Contact** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Edit the **Title Text** and the **Body Text** (the rich-text intro, including the contact email).</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Adjust the input placeholders (Name, Email, Subject, Message) and the **Submit Button Text**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "Changing the contact email"
    The email address shoppers are told to write to lives inside the **Body Text**
    field — edit it there.

# Editing the Discount Code Popup

<span class="bla-badge">Popup</span>

**What it changes:** The gift/discount-code popup that appears after a short delay —
whether it's on, how long before it shows, and the cancel-button wording and colors.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → click
**Code Popup** in the left sidebar.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Select the **Code Popup** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Use **Is Enabled?** to turn the popup on or off, and set the **Popup Appear Delay (S)**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Edit the **Cancel Button** text and colors. (The popup's main accent/text colors are in **Theme settings → Code Popup Settings**.)</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "It won't show on every page"
    The popup is intentionally suppressed on the homepage and on the Signup / Restock
    Alert pages, so it never interrupts those flows.

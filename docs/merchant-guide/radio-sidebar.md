# Editing the Basement Radio Panel

<span class="bla-badge">Radio</span>

**What it changes:** The slide-out Basement Radio (audio) panel — its colors, the
mobile exit-arrow label, and the fallback image. (The tracks themselves are blocks.)

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → click the
**Basement Radio Sidebar** section in the left sidebar.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Select the **Basement Radio Sidebar** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Set the **Primary**, **Secondary**, **Background**, and **Button Selected** colors.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Upload a **Radio Default Image** (shown if a track's artwork fails to load), and edit tracks via the section's blocks.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

# Editing the Header / Navigation

<span class="bla-badge">Header</span>

**What it changes:** The top navigation bar — its colors, whether it has a background,
the shop drawer tiles, and the restock-alert link. (The menu **links** are set in
Theme settings.)

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → open the
**Header** group in the left sidebar → click **Header Menu**.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open the **Header** group and select **Header Menu**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Set the **Text Color** and **Does Header Have Background?** to suit the page background.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Edit the shop tiles (images + links) via the **Header Shop Item** blocks inside the drawer/sidebar.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "Menu links live in Theme settings"
    The actual navigation links (left menu, right menu, sidebar menus) are chosen in
    **Theme settings → Header Menu** and **Header Sidebar**, and the links themselves
    are built under **Content → Menus**.

!!! note "Two header versions"
    The homepage and cover pages use a "no background" header so the hero shows
    through. If a change doesn't appear there, edit that page's header group too.

# Editing the Password (Coming Soon) Page

<span class="bla-badge">Pages</span>

**What it changes:** The password-protected "New Drop" page shown while the store is
locked — the title, countdown, image carousel, and the password/subscribe buttons.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template (top bar) to **Password** → click the **Password** section.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">In the template dropdown choose **Password**, then select the **Password** section. Temporarily tick **Show Password Page** so you can see it while editing.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Set the **Title Text**, the **Date and Time Before Drop** (`mm/dd/yyyy hh:mm:ss`), and the **Countdown Done Text**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Add carousel images via the section's blocks, and paste your **Klaviyo Form ID** for the subscribe form.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">**Untick Show Password Page**, then click **Save**.</span>
</div>

!!! warning "Turning the lock on/off is separate"
    The **Show Password Page** toggle here is only for previewing in the editor. The
    real store lock is controlled in **Online Store → Preferences → Password page**.
    Leave **Show Password Page** disabled before saving.

!!! note "Carousel speed"
    Keep **Img Carousel Speed** a whole number (no decimals) — decimals can break the
    scroll in Safari.

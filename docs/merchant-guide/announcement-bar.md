# Editing the Announcement Bar

<span class="bla-badge">Header</span>

**What it changes:** The thin promo strip at the very top of every page (e.g. "FREE
SHIPPING ON US ORDERS 175+"), including its optional countdown timer.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → in the left
sidebar open the **Header** group → click **Announcement Bar**.

## Editing the Text

![Editing the announcement text](../assets/gifs/announcement-bar-text.gif)

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open the **Header** group and select **Announcement Bar**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Edit **Announcement Text** and adjust the text/background colors.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

<!-- !!! note "Two versions exist"
    The homepage and cover pages use a separate "no background" header, which has its
    own announcement bar copy. If your edit doesn't show on one of those pages, edit
    that page's header group too. -->

## Adding a Countdown

![Adding a countdown to the announcement bar](../assets/gifs/announcement-bar-countdown.gif)

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open the **Header** group and select **Announcement Bar**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Tick **Is Countdown Enabled?**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Set **Date and Time Before Drop** using the exact format `mm/dd/yyyy hh:mm:ss` (e.g. `11/30/2025 23:59:00`).</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Adjust the **Time Block** text/background colors if needed, then click **Save**.</span>
</div>

!!! warning "Date format matters"
    The countdown date is a plain text field — it must be typed exactly as
    `mm/dd/yyyy hh:mm:ss`. A mistyped date will make the timer blank or stuck.

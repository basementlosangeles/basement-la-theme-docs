# Editing the 404 (Page Not Found) Page

<span class="bla-badge">Pages</span>

**What it changes:** The headline and the "shop" button shoppers see when they hit a
broken or missing URL.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template (top bar) to **404 page** → click the **404 Page** section in the left
sidebar.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">In the template dropdown at the top of the editor, choose **404 page**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Select the **404 Page** section and edit **Title Text** (default "Page Not Found").</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Set the **Shop Button Text**, **Link**, and the text/background colors.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! tip "Test it"
    Visit any made-up URL on the live site (e.g. `/this-page-does-not-exist`) to see
    your changes in context.

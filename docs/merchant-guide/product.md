# Editing Product Pages

<span class="bla-badge">Product</span>

**What it changes:** Shared settings for every product page — navigation button colors,
the preorder message, and the fallback text shown when a product is missing details, a
size chart, or terms.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template to a **Product** page → click the **Product** section.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open a **Product** template and select the **Product** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Edit the **Preorder Text** and **Preorder Tag Title** — products tagged with that title will show the preorder message.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Adjust the "Not Available" fallback texts (No Details / No Size Chart / No Terms) if needed.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "Per-product content comes from the product itself"
    Images, description, color swatches, and the size chart are set on each product in
    **Shopify Admin → Products** (via product fields and metafields), not in this
    section. This section only controls the shared layout and fallback wording.

!!! warning "Leave the preview toggle off"
    **Show Desktop Fullscreen Image View?** is for editor preview only — keep it
    disabled for production.

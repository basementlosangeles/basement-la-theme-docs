# Adding Spacing Between Sections

<span class="bla-badge">Layout</span>

**What it changes:** Adds empty vertical space between sections. Useful for breathing
room without touching code.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → on any page,
click **Add section** → choose **Spacer**.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Click **Add section** where you want the gap, and choose **Spacer**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Set **Spacing (PX)** for a single value across all devices.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Need different gaps on mobile vs desktop? Tick **Has Mobile and Desktop Variants?** and set **Mobile Spacing** and **Desktop Spacing** separately.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Drag the Spacer into position and click **Save**.</span>
</div>

# Editing the Signup / Restock Alert Page

<span class="bla-badge">Pages</span>

**What it changes:** The "New Drop Coming Soon" signup page (also used for the Restock
Alert page) — its heading, subtitle, optional image, optional countdown, and the
connected Klaviyo form.

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → switch the
template to the **Signup** page (or **Restock Alert** page) → click the **Signup**
section.

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Open the **Signup** (or **Restock Alert**) page in the template dropdown and select the **Signup** section.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Paste the **Klaviyo Form ID**, then edit the **Title Text** and **Subtitle Text**.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Optionally toggle **Show Image** / **Show Countdown** and set the **Date and Time Before Drop** (`mm/dd/yyyy hh:mm:ss`).</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! note "Same section, two pages"
    The Signup page and the Restock Alert page both use this section, but each keeps
    its own settings — update the one whose template you're editing.

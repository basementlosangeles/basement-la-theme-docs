# Editing the Homepage Hero

<span class="bla-badge">Homepage</span>

**What it changes:** The full-screen hero at the top of the homepage — its background
image or video, the overlay/gradient, and the "Enter Basement" button.

!!! warning "Confirm this section is in use first"
    The current homepage is built from the **Collection** grid section, and this
    **Homepage** hero section may not be active on the live site. Before relying on
    this guide, confirm with a developer whether the Homepage section is assigned to
    the homepage template. See the
    [developer note](../developer-guide/homepage.md).

**Where to find it:** Shopify Admin → Online Store → Themes → Customize → **Home page**
template → click the **Homepage** section (if present).

## Steps

<div class="bla-step" markdown>
<span class="bla-step__num">01</span>
<span class="bla-step__body">Select the **Homepage** section and choose **Media Type** — Image or Video.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">02</span>
<span class="bla-step__body">Upload the **Mobile** and **Desktop** background image (or video), then nudge the X/Y position and zoom so the focal point sits right on each device.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">03</span>
<span class="bla-step__body">Adjust the **Background Overlay** / gradient, then set the **Enter Button** text, color, and link.</span>
</div>

<div class="bla-step" markdown>
<span class="bla-step__num">04</span>
<span class="bla-step__body">Click **Save**.</span>
</div>

!!! warning "Compress large media"
    Oversized hero images/videos slow the homepage down. Compress images before
    uploading and keep background video short and light.
